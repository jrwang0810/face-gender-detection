# Deploy OpenCV and Keras Models with A Flask Web App: Face Detection and Gender Classification

## Getting Started in 10 Minutes

- Clone this repo
- Install requirements
- Run the script
  python app.py
- Go to http://127.0.0.1:5002/
- Done! :tada:
